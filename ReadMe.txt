Heya!
	This project analyzis some (100) latest tweets from the given user account for negative or positive sentiments.
	
IMPORTS:
	
	1. tweepy: 	     Twitter authentication API
	2. NLTK:	     This is an ML library that breaks the content of the web-page into a list of sentences (sentence tokenize)
	3. TextBlob:	     This AI library helps to understand positive and negative sentiments behind some text input.
				  like:- The story and plot was amazing.     : A positive review ( polarity > 0 )
				         An awful movie.		     : A negative review ( polarity < 0 )
				         The movie has 4 songs.		     : A neutral review  ( polarity = 0 )

	4. Pandas: 	     To set the results into a DataFrame.
	5. MatPlotLib: 	     To represent the result graphicaly. ( Used PIE chart and Scatter graph. )
	6. FigureCanvasTkagg:To plot the graph on Tkinter GUI Window.
	7. Re:		     Stands for regular expression. Used for tweets data filtering.
	8. PIL:		     Or Pillow for ImageHandling on Tkinter Screen.		    

Go through the code. And run on any PYTHON3 IDE/Console
